Tag Cloud for Octopress Description: Easy output tag cloud and category list.

Files: . ├─ plugins/ │ └── tag_cloud.rb └─ source/ └─ _includes/ └─ custom/ └─ asides/ ├─ category_list.html └─ tag_cloud.html Syntax: ios(2) other(14) web(23)

  • ios (2)
  • other (14)
  • web (23)

    • Example: In some template files, you can add the following markups.

    source/_includes/custom/asides/tag_cloud.html

    Tag Cloud

    ios other web

    source/_includes/custom/asides/category_list.html

    Categories

    • ios (2)
    • other (14)
    • web (23)

    Notes: Be sure to insert above template files into default_asides array in _config.yml. And also you can define styles for ‘tag-cloud’ or ‘category-list’ in a .scss file. (ex: sass/custom/_styles.scss)
